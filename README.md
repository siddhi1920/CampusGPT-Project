📄 CampusGPT – AI-Based Academic Assistant

CampusGPT is an AI-powered academic assistant designed to help students interact with study materials efficiently. The application allows users to upload PDF documents and ask questions based on the content, providing accurate and context-aware answers using advanced Natural Language Processing (NLP) techniques.

The system supports two modes: Offline Mode, where answers are generated from uploaded PDFs using vector similarity search, and Online Mode, where general questions are answered using an AI language model. CampusGPT uses FAISS for fast document retrieval, HuggingFace embeddings for text representation, and integrates with modern AI models for response generation.

The platform includes a secure user authentication system, chat history management, and an admin dashboard for monitoring user activity and system usage. Built using Python and Streamlit, CampusGPT provides an intuitive and interactive interface for seamless user experience.
